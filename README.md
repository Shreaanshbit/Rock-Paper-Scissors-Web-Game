# Rock Paper Scissors Web Game

A simple, interactive web-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## How to Play
- Choose rock, paper, or scissors by clicking the corresponding button.
- The computer will randomly select its choice.
- The game determines the winner based on standard rules:
  - Rock beats scissors.
  - Paper beats rock.
  - Scissors beats paper.
- Scores are tracked for you and the computer, along with the number of attempts.

## Features
- Real-time score updates.
- Displays the computer's choice after each round.
- Fun emojis for win/lose messages.

## Files
- `index.html`: Main HTML structure.
- `style.css`: Styling for the game interface.
- `script.js`: Game logic and interactivity.

## How to Run
- Open `index.html` in any modern web browser.
- You can also use the link  https://rockpaperscissorsbyshreaansh.netlify.app/
